Aprendiendo - Suma Algebraica

Caso1: signos iguales

Signos iguales se suman y se coloca el mismo signo

-a -b = -(a+b)

Ejemplos:

1. -2-4 = -(2+4) = -6

2. -6-1 = -(6+1) = -7

3. -5-0 = -(5+0) = -5

4. -0-0 = -(0+0) = -0 = 0 ; El cero no tiene signo definido, puede ser negativo (-) o positivo (+)
